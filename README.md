<!-- Run Locally -->

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/sagnikghosh1999/facebook-clone-frontend.git
```

change directory

```bash
  cd facebook-clone-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
